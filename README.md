# legal_advisor

Code and project notes for the `Legal_advisor` analysis workspace.

## Tracked in this repository

- `code/`: analysis and data-processing scripts
- top-level project memos in Markdown

## Not tracked in this repository

- `data/`: raw, intermediate, and output datasets
- `output/`: generated figures and tables
- `_archive/`: archived inputs and backups

This repository is configured as a code-only sync target so the local research data can remain in Dropbox without being pushed to GitHub.
